## Exp_05 - Php code to List all assistants and Delete an assistant
<br>

### Objective
- Create a simple php web app that connects to the OpenAi API
- Create an assistant
- Create the Php code to list all assistants
- Create Php code to Delete an assistant by entering the assistant id

### Notes
- A list of all created assistants can also be viewed on the OpenAi site:<br>
https://platform.openai.com/assistants
- The above link will show all created assistants and their ids. When an asistant is deleted, this list gets updated.
